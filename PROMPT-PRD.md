# 블로그 라이터 프롬프트 PRD — 네이버 상위노출 글쓰기 공식

## 문서 목적
네이버 검색 1위 뷰티샵 블로그 글 8개를 분석하여 추출한 공통 패턴.
이 문서를 기반으로 `seo-prompts.ts`의 SEO_REVIEW_STYLE_PROMPT를 개선한다.

## 2. 핵심 발견: 현재 프롬프트 vs 실제 1위 글

### 현재 프롬프트 (원장 전문가 시점)
```
"이런 분들이 오시거든요"
"현장에서 보면요"
"업계에서 보면"
→ 정보 전달형, 판단 기준 제시형
```

### 실제 1위 글 (고객 체험 후기 시점)
```
"회사 회식 자리에서 동료 언니 손톱이 반짝이는 게 보였거든요"
"블랙으로 덮었던 걸 빼야 하는 상황이라"
"거울 볼 때마다 기분이 좀 다운됐거든요"
→ 스토리텔링형, 감정 공감형, 체험 후기형
```

### 결론
네이버 상위노출 뷰티샵 글의 본질은 **"고객이 직접 쓴 것 같은 체험 후기"** 스타일이다.
원장이 쓰더라도, 글의 화자는 **체험자/방문자** 시점이어야 한다.

## 3. 상위노출 글의 공통 구조 (8개 글 공통)

### 3-1. 감정 흐름 공식 (필수)
```
고민/불만 → 검색 → 발견 → 기대 → 방문 → 감동 → 만족 → 추천
```

모든 1위 글이 이 흐름을 따른다. 순서를 벗어나지 않는다.

### 3-2. 글 구조 템플릿

```
[도입부] 일상 속 고민 시작 (2~3문단)
  - "회사 회식에서 동료 손톱이 반짝이는 게 보였거든요"
  - "머릿결이 푸석해져서 관리가 필요하다고 느꼈어요"
  - "헬스장에서 운동하다 보니 위생관리에 신경이 쓰이더라고요"
  ※ 키워드 1회 삽입

[편집가이드: 대표 이미지 or 시술 결과 사진]

[업체 정보 블록] 이름 + 주소 + 지도
  - 네이버 플레이스 지도 삽입 위치
  - 영업시간, 주차정보, 전화번호
  ※ 글 초반~중반에 배치

[편집가이드: 매장 외관/간판 사진]

[매장 분위기] 인테리어 묘사 (1~2문단)
  - "화이트톤 인테리어가 눈에 확 들어왔어요"
  - "깔끔하면서도 따뜻한 느낌"
  - 위생/청결 강조

[편집가이드: 매장 내부 인테리어 사진 2~3장]

[상담 과정] 꼼꼼한 상담 강조 (1~2문단)
  - 시술자 대화 직접 인용
  - "과한 시술 권유 없이" 신뢰 키워드
  - 사용 제품/브랜드 언급
  ※ 키워드 1회 삽입

[편집가이드: 상담 장면 or 제품 사진]

[시술 과정] 단계별 서술 (3~5단계, 핵심 구간)
  - 각 단계마다 감각 묘사 + 시술자 대화 인용
  - "두껍게 바르면 금방 들뜨니까 얇게 여러 번 중첩하는 게 포인트예요"
  - "브러시로 살살 쓸어내리시는데, 마치 구름이 퍼지듯"
  ※ 키워드 2~3회 분산 삽입

[편집가이드: 시술 과정 사진 (단계별)]

[완성/결과] 비포-애프터 (1~2문단)
  - 감탄 표현: "거울 보는 순간 입이 떡 벌어졌어요"
  - 구체적 변화 묘사

[편집가이드: 시술 전후 비교 사진]
[편집가이드: 다양한 각도에서 찍은 결과 사진 2~3장]

[애프터케어] 관리 방법 안내 (1문단)
  - 시술 후 주의사항
  - 실용적 정보 (독자가 저장/공유할 이유)

[마무리] 추천 멘트 (1~2문단)
  - "정착할 생각이에요"
  - "다음엔 어떤 디자인 할지 벌써 기대돼요"
  ※ 키워드 마지막 1회 삽입

[편집가이드: 매장 정보 + 네이버 지도 삽입]
```

## 4. 문체 규칙 (8개 글 공통 패턴)

### 4-1. 문장 규칙
| 항목 | 규칙 | 예시 |
|------|------|------|
| 문장 길이 | 2~3줄 이내, 짧게 끊기 | O: "정말 놀랐어요.\n문을 열자마자 화이트 톤 인테리어가 눈에 확 들어왔거든요." |
| 줄바꿈 | 문단마다 빈 줄(​) 넣기 | 모든 1위 글이 2~3줄마다 빈 줄 삽입 |
| 말투 | ~요/~죠/~거든요/~더라고요 | "검색하기 시작했죠", "기대됐거든요" |
| 대화체 | 시술자 말 직접 인용 (큰따옴표) | "큐티클 케어부터 해드릴게요. 건강한 베이스가 있어야 네일이 더 오래가요." |
| 감성 표현 | 비유, 감각 묘사 활용 | "구름이 퍼지듯 자연스러운 그라데이션", "별처럼 반짝이고" |
| 이모티콘 | 적당히 사용 (ㅎㅎ, ㅠㅠ 정도) | "남편한테 자랑하려고 찍은 사진 ㅎㅎ" |

### 4-2. 도입부 패턴 (실제 1위 글 기반)
절대 "안녕하세요 오늘은~"으로 시작하지 않는다.

**패턴 A — 일상 에피소드 시작**
```
"회사 회식 자리에서였어요. 동료 언니가 잔을 들 때마다 손톱이 반짝이는 게 보였거든요."
```

**패턴 B — 고민/불만 시작**
```
"요즘 머릿결이 푸석해져서 관리가 필요하다고 느꼈어요."
```

**패턴 C — 계기 시작**
```
"삿포로로 여행 떠나기 전 봄 내음 가득 새로운 네일아트를 받고 왔어요."
```

**패턴 D — 상태 묘사 시작**
```
"요즘 손끝이 너무 초라했어요. 일만 하다 보니 관리할 시간도 없었고."
```

**패턴 E — 동기 시작**
```
"헬스장에서 운동을 꾸준히 하다 보니, 위생관리에 신경이 더 많이 쓰이더라고요."
```

### 4-3. 시술자 대화 인용 패턴
1위 글들은 시술 과정에서 시술자(원장/선생님)의 말을 직접 인용한다.
이것이 전문성과 신뢰감을 동시에 만든다.

```
"큐티클 케어부터 해드릴게요. 건강한 베이스가 있어야 네일이 더 오래가요."
"두껍게 바르면 금방 들뜨니까 얇게 여러 번 중첩하는 게 포인트예요."
"너무 많으면 부담스러우니까 적당히만 포인트 줄게요."
"뜨거운 물 조심하시고, 무리하게 제거하지 마세요."
"혹시 불편한 데 없으세요?"
```

→ 본문에 최소 2~3개의 시술자 대화 인용을 포함해야 한다.

### 4-4. 마무리 패턴
```
"정착할 생각이에요" (재방문 의사)
"다음엔 어떤 디자인 할지 벌써 기대돼요" (기대감)
"재방문 의향 충분히 있고" (확신)
"무조건 추천드리고 싶은 곳이에요" (추천)
```

## 5. SEO 규칙 (8개 글 공통)

### 5-1. 키워드 배치
- 제목: 키워드 1회 (맨 앞에 배치)
- 본문 첫 2~3문장: 1회
- 본문 중간 (시술 과정 구간): 2~3회 분산
- 본문 마지막 2~3문장: 1회
- **합계: 5~7회** (자연스럽게)
- 대형 키워드(시/구 단위): 5~7회
- 소형 키워드(동 단위): 4~6회

### 5-2. 이미지 삽입 (체류시간 핵심)
- 최소 10장 이상의 사진 삽입 위치를 편집가이드로 안내
- 사진이 많을수록 체류시간 증가 → 상위노출에 직결
- 편집가이드 삽입 위치:
  - 대표 이미지 (글 최상단)
  - 업체 외관/간판
  - 매장 내부 인테리어 (2~3장)
  - 상담 장면 or 제품
  - 시술 과정 (단계별 3~5장)
  - 완성 결과 (다양한 각도 2~3장)
  - 비포/애프터 비교
  - 매장 정보 + 지도

### 5-3. 글 길이
- 최소 2,000자 이상 (편집가이드 제외)
- 이상적: 2,500~3,500자
- 짧은 문단 + 빈 줄이 많아서 실제 스크롤은 매우 길어짐

### 5-4. 업체 정보 배치
- **글 초반~중반**: 업체명 + 주소 + 네이버 지도
- **글 하단**: 영업시간, 주차, 연락처 (선택)
- 지도는 네이버 플레이스 연동

### 5-5. 제목 공식
```
[지역명] + [업종 키워드] + [구체적 시술/서비스] + [후기성 멘트]
```
예시:
- "청주네일아트, 벨르뮤 밀키 글리터 인생 네일 찾음"
- "천안 미용실 블랙 빼고 깔끔하게 염색한 후기"
- "강남두피관리 생각보다 좋았던 후기"
- "천안 속눈썹펌 잘하는 곳 두정동 내것처럼 내 눈매 맞춤형 컬링"

## 6. 신뢰 요소 패턴 (1위 글들이 공통으로 사용)

### 6-1. 전문성 어필
- 수상 경력, 자격증 언급
- 사용 제품 브랜드 노출 (아베다, 정품 등)
- "아모스 탈색, 염색 컬러 헤어쇼에서 전국 1등"
- "아베다 공식 파트너 살롱"
- "식약처 인증", "환경부 인증"

### 6-2. 과한 권유 없음 강조
- "과한 시술 권유 없이 진행된 점이 좋았어요"
- "필요한 것만 추천해주셔서 믿음이 갔어요"
- "불필요한 작업은 하지 않으면서"

### 6-3. 위생/청결 강조
- "일회용 위생 시트는 기본"
- "갓 오픈한 새 매장처럼 깨끗"
- "수건이나 침구류도 뽀송뽀송하게 준비"

### 6-4. 프라이빗/1인실
- "1인 전용 프라이빗 룸"
- "예약제로 운영되어 다른 손님과 겹치지 않아"
- "독립된 공간에서 편안하게"

## 7. 업종별 톤 차이 (분석 결과)

### 네일
- 가장 감성적, 여성스러운 톤
- "여리여리", "뽀용", "영롱" 등 감각 표현 다수
- 디자인 선택 과정 상세 묘사

### 헤어(미용실)
- 실용적이고 담백한 톤
- 시술 과정을 기술적으로 상세히
- 비포/애프터 변화 강조

### 에스테틱/피부관리
- 가장 캐주얼하고 친근한 톤 (ㅋㅋ, ㅠㅠ, ㅎㅎ 다수)
- 본인 피부 고민 상세 서술 → 공감 유도
- 시술 루틴 전체 공개

### 두피관리
- 에스테틱과 유사하지만 더 상세한 과정 묘사
- 이모지 활용 (소제목에 이모지)
- 이벤트/할인 정보 포함

### 속눈썹
- 전문적이면서도 꼼꼼한 톤
- 비포/애프터 강조 (눈매 비대칭 교정 등)
- 유지력/지속기간 후기

### 왁싱
- 프라이빗/위생 가장 강조
- 첫 경험자 타겟 ("처음이라 떨렸지만")
- 통증/불편함에 대한 솔직한 후기

## 8. 민감 단어 우회 패턴

네이버 검색 필터를 우회하기 위해 1위 글들이 사용하는 패턴:
```
시술 → 시.술.
아토피 → 아.토.피.
염증 → 염.증.
보톡스 → 보톡*
```

→ 프롬프트에서 직접 우회 처리하지 않고, 
편집가이드로 "민감 단어는 가운데점(.)으로 분리해주세요" 안내.

## 9. 소제목 스타일 옵션

1위 글들이 사용하는 소제목 스타일 (랜덤 선택):

| 스타일 | 예시 |
|--------|------|
| 소제목 없음 | 줄바꿈만으로 구분 (가장 자연스러움) |
| 볼드 텍스트 | **한결 팀장님 상담이 신뢰감 있었어요** |
| 체크박스 | ☑️ 강남눈썹반영구, 어디서 받아야 할까 |
| 넘버링 | 1. 위치와 첫인상 / 2. 맞춤 상담 |
| 이모지 소제목 | 🔍 내 두피의 민낯을 마주하다 |
| 키워드 반복형 | "강남 미용실 위치 접근성 괜찮았어요" |

## 10. 프롬프트 적용 방향

### AS-IS (현재 SEO_REVIEW_STYLE_PROMPT)
```
- 원장/시술자 시점 ("이런 분들이 오시거든요")
- 정보 전달 + 판단 기준 제시
- 전문가 블로그체
```

### TO-BE (개선 방향)
```
- 고객/체험자 시점 ("검색하다가 후기 좋은 곳을 발견했어요")
- 스토리텔링 + 감정 공감 + 시술 과정 상세 묘사
- 체험 후기체 (자연스러운 블로그 글)
- 시술자 대화 직접 인용 2~3회 필수
- 편집가이드로 사진 삽입 위치 10곳 이상 안내
```

### 주의: 두 스타일 모두 유지
원장 전문가 시점도 유효한 글쓰기 방식이므로 삭제하지 않는다.
사용자가 선택할 수 있도록 두 가지 모드를 제공:

1. **체험 후기형** (신규 추가) — 상위노출 최적화
2. **전문가 정보형** (기존 유지) — 원장 브랜딩

## 11. 체험 후기형 프롬프트 핵심 규칙

```
[필수 요소]
1. 화자는 "체험자/방문자" — 1인칭 자연스럽게 사용
2. 감정 흐름: 고민 → 검색 → 발견 → 방문 → 시술 → 결과 → 추천
3. 시술자 대화 직접 인용 최소 2~3회
4. 짧은 문장 (2~3줄) + 빈 줄로 가독성 극대화
5. 편집가이드 10곳 이상 (사진 삽입 위치)
6. 업체 정보(이름, 주소, 지도) 글 초반~중반에 배치
7. 애프터케어/관리법으로 마무리
8. 키워드 5~7회 자연스럽게 분산

[금지 요소]
1. "안녕하세요 오늘은~" 시작 금지
2. 마크다운(##, **, ---) 금지
3. 의료 용어 직접 사용 금지
4. 영업 유도 (예약하세요, 문의주세요) 금지
5. 감정 과장 (대만족, 감동, 감탄) 금지
6. 업체명을 반복적으로 노출하는 것 금지 (1~2회만)
7. 백과사전식 정보 나열 금지
```
